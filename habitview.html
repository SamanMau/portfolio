HabitView - Track Your Lifestyle Habits

What is HabitView?

HabitView is a web-based application designed to help users track and visualize habits related to sleep, diet, and exercise. Users receive AI-generated recommendations via ChatGPT to improve their lifestyle, and dynamic graphs visualize their progress clearly.

Key Features:

  • Dynamic Data Visualization: Interactive graphs to track progress.
  • AI Recommendations: Personalized feedback powered by OpenAI GPT.
  • PDF Reports: Export user progress as detailed PDF reports.
  • User Verification: Secure registration and email verification.
  • Technologies Used: Java, Spring Boot, PostgreSQL, HTML, CSS, JavaScript, CI/CD.
